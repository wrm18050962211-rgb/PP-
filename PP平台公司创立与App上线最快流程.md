# PP 平台公司创立与 App 上线最快流程

本文档按“最快把公司和业务跑起来”的顺序写，目标不是一开始把所有证照和复杂合规一次性做满，而是先建立合法经营主体、跑通小范围订单、验证抽佣模型，再逐步补齐后续资质和制度。

适用业务：PP 陪拍平台，核心模式为内容展示、本地生活服务撮合、用户预约陪拍者、平台抽佣、后续 App/小程序上线。

## 总路线

最快启动路线：

```text
找杨浦可注册地址/小工位
-> 注册网络科技公司
-> 开银行基本户和税务
-> 找代理记账
-> 先做微信小程序 MVP
-> 接入微信支付
-> 准备最简协议
-> 招募首批陪拍者
-> 小范围试运营
-> 跑通抽佣和结算
```

后补路线：

```text
ICP备案 / 小程序备案 / App备案
-> 支付分账
-> 认证和风控加强
-> 商标注册
-> EDI/ICP经营许可证咨询
-> App 正式上架
-> 政策补贴和园区申报
```

## 第一阶段：最快启动必须做

### 1. 确定公司名称

优先使用网络科技、信息科技、数字科技方向，不要一开始用“陪伴”“交友”“人力资源”“旅行社”等容易引起额外监管联想的名称。

推荐名称结构：

```text
上海XX网络科技有限公司
上海XX信息科技有限公司
上海XX数字科技有限公司
```

准备 3-5 个备选名称，因为系统核名可能重名或近似。

命名建议：

- 名字不要太长，方便用户记住。
- 尽量能做品牌名和 App 名。
- 先查微信小程序、App Store、商标、域名是否被占用。
- 如果未来要融资，名称要显得像技术平台，而不是个人工作室。

不建议名称：

```text
上海XX陪伴服务有限公司
上海XX交友服务有限公司
上海XX摄影工作室有限公司
上海XX人力资源有限公司
上海XX旅游服务有限公司
```

原因：这些名字容易把业务导向陪侍、婚恋、劳务中介、旅行社、传统摄影店等方向，后续平台审核、支付审核、应用市场审核可能更麻烦。

### 2. 确定公司基本结构

注册前先定下面这些信息。

公司类型：

```text
有限责任公司
```

如果只有你一个人：

```text
一人有限责任公司
```

如果有合伙人：

```text
普通有限责任公司，按股权比例登记
```

需要提前确认：

- 谁当法定代表人。
- 谁当股东。
- 每个股东占多少股份。
- 注册资本写多少。
- 谁当财务负责人。
- 是否需要监事，按登记系统要求执行。
- 公司联系电话和邮箱用哪个。

早期注册资本建议：

```text
10 万 - 100 万认缴
```

不建议为了面子写几百万或上千万。注册资本是股东承诺承担责任的边界之一，写太高没有必要。

如果有合伙人，一定在注册前写清楚：

- 谁出钱。
- 谁出技术。
- 谁负责运营。
- 谁负责商务。
- 如果有人退出怎么办。
- 股权是否分期释放。
- 公司亏损怎么承担。
- 后续融资股权怎么稀释。

### 3. 找杨浦可注册的小型办公场地

你的优先目标不是大办公室，而是：

```text
杨浦区园区/孵化器/联合办公 + 1-3 个工位 + 可工商注册
```

优先区域：

- 五角场
- 大学路
- 创智天地
- 长阳创谷
- 杨浦滨江附近创业园

你看场地时直接问这些问题：

1. 能不能用于工商注册？
2. 是否提供真实租赁合同？
3. 是否提供房产证复印件或登记所需住所材料？
4. 是否能配合银行开户上门核验？
5. 是否能配合税务核查？
6. 地址是否是正规集中登记地或登记电子地图可核验地址？
7. 是否提供创业政策申报辅导？
8. 是否能申请租金补贴、创业培训、创业担保贷款等政策？
9. 是否要求税务必须落在杨浦？
10. 后续迁址是否收费？
11. 工位能否月付或季付？
12. 最短租期多久？

早期成本目标：

```text
1 个开放工位：每月几百到一千多
小独立间：每月两三千起，视位置而定
```

不建议一开始：

- 租 50 平以上办公室。
- 年付大额租金。
- 用特别便宜但无法开户、无法税务核查的虚拟地址。
- 用住宅地址硬注册。

判断一个地址是否适合你的标准：

```text
能注册
能开户
能税务核查
能收信件
能后续申请政策
成本低
交通不太偏
```

### 4. 准备经营范围

你可以把公司包装成“数字文旅、本地生活服务撮合、内容社区、预约平台、软件技术服务”。

建议经营范围包含：

```text
软件开发；
网络技术服务；
信息技术咨询服务；
互联网数据服务；
数字内容制作服务；
广告设计、代理；
广告发布；
市场营销策划；
组织文化艺术交流活动；
摄影扩印服务；
摄像及视频制作服务；
居民日常生活服务；
商务代理代办服务；
互联网销售；
企业管理咨询；
会议及展览服务。
```

根据登记系统可选项微调，不一定逐字一样。

先不要主动写：

```text
劳务派遣
人力资源服务
职业中介
婚姻介绍
旅行社业务
金融服务
支付结算
互联网新闻信息服务
网络文化经营
```

这些通常涉及许可或更高审核要求，与你早期 MVP 无关。

### 5. 在线提交公司设立

办理入口：

```text
上海企业登记在线
https://yct.sh.gov.cn/portal_yct/
```

提交前准备：

- 公司名称备选。
- 法人身份证信息。
- 股东身份证信息。
- 财务负责人信息。
- 注册地址材料。
- 注册资本。
- 股权比例。
- 经营范围。
- 联系电话。
- 联系邮箱。

线上流程大致是：

```text
名称自主申报
-> 填写设立信息
-> 上传/填写住所材料
-> 填写股东和高管信息
-> 填写经营范围
-> 提交预审
-> 法人和股东实名认证
-> 电子签名
-> 等待审核
-> 领取营业执照
```

注意事项：

- 手机号要保持可用，后续税务、银行、工商都会用。
- 法人、股东要能配合电子签名。
- 地址材料如果由园区提供，让园区确认填写口径。
- 如果系统退回，不要反复乱改，直接问园区或登记窗口原因。

### 6. 领取营业执照和印章

公司设立通过后，你要拿到：

- 营业执照。
- 公章。
- 财务章。
- 法人章。
- 发票章。

这些东西用途：

- 公章：签合同。
- 财务章：银行和财务。
- 法人章：银行开户等。
- 发票章：开票。
- 营业执照：平台认证、银行开户、支付商户认证、应用市场认证。

保管原则：

- 不要把公章随便交给不可信的人。
- 不要让代理长期无理由保管全部章。
- 每次盖章要留记录。
- 合同盖章前先看清楚付款、违约、期限和退出条款。

### 7. 开银行基本户

营业执照和章拿到后，尽快开公司银行基本户。

准备材料通常包括：

- 营业执照。
- 公章、财务章、法人章。
- 法人身份证。
- 公司章程或登记材料。
- 租赁合同或注册地址材料。
- 实际办公场地照片，有些银行会看。

银行可能会核查：

- 公司是否真实经营。
- 办公地址是否真实。
- 法人是否本人到场。
- 公司业务是否说得清楚。

你对银行解释业务时可以这样说：

```text
我们做一个本地生活预约平台，主要面向城市拍照、探店、Citywalk 等场景，用户可以预约本地摄影陪拍服务者。公司收入来自平台技术服务费和订单服务费。
```

不要说：

```text
陪人出去玩、交友、线下陪伴、私人见面、代收款再转账
```

这些表达容易让银行误解你的风险。

### 8. 找代理记账并完成税务

第一次创业，建议直接找代理记账。

你要让代理记账帮你做：

- 税务登记。
- 税种核定。
- 每月记账报税。
- 个税申报。
- 发票申请。
- 工商年报提醒。
- 汇算清缴提醒。

选择代理记账时问：

1. 每月多少钱？
2. 是否包含零申报？
3. 是否包含个税申报？
4. 是否包含工商年报？
5. 是否协助申请发票？
6. 是否提醒社保、公积金、汇算清缴？
7. 如果平台有抽佣收入，如何入账？
8. 如果给陪拍者结算，凭证怎么处理？

早期税务原则：

- 不要用个人卡收平台订单钱。
- 公司收入进入公司账户或公司支付商户号。
- 每一笔平台佣金要能解释。
- 用户退款、陪拍者结算、平台补贴要分开记录。

### 9. 先做微信小程序 MVP

为了最快上线，不建议第一版直接做完整 App。

优先做：

```text
微信小程序 MVP
```

原因：

- 登录快。
- 支付接入快。
- 分享传播方便。
- 审核通常比原生 App 路线轻。
- 适合小范围试运营。
- 不需要用户下载安装。

第一版小程序只做最必要功能：

- 首页图片流。
- 陪拍者列表或作品列表。
- 帖子详情。
- 预约按钮。
- 下单支付。
- 我的订单。
- 陪拍者入驻申请。
- 客服/举报入口。

第一版先不要做：

- 复杂推荐算法。
- 完整社区关注体系。
- IM 私信大系统。
- 大规模内容发布。
- 会员体系。
- 多城市扩张。
- 太复杂的优惠券系统。

### 10. 开通微信支付商户号

你要用公司主体申请微信支付商户号。

准备材料：

- 营业执照。
- 法人身份证。
- 公司银行账户。
- 小程序 AppID。
- 客服电话。
- 经营类目说明。
- 网站/小程序页面截图。

经营类目尽量按本地生活、摄影服务、生活服务、互联网服务等方向选择，具体以微信支付后台可选类目为准。

早期收款模型：

```text
用户支付订单
-> 款项进入公司微信支付商户
-> 平台记录订单金额、佣金、应结算金额
-> 服务完成后结算给陪拍者
```

注意：

- 可以先用简单结算跑小规模订单。
- 不要长期大额沉淀陪拍者资金。
- 不要走个人微信收款码。
- 不要让用户直接转账给陪拍者，否则平台无法抽佣，也无法处理纠纷。

### 11. 准备最简协议

小范围试运营前，至少准备 4 份协议。

#### 用户协议

要写清：

- 平台提供什么服务。
- 用户如何预约。
- 用户不得私下交易。
- 用户不得发布违法、骚扰、侵犯隐私内容。
- 用户应遵守线下安全规则。
- 平台可以处理违规账号。

#### 隐私政策

要写清：

- 收集哪些信息。
- 为什么收集。
- 如何保存。
- 是否给第三方支付、云服务、地图服务使用。
- 用户如何注销账号。
- 用户如何联系客服。

#### 陪拍者入驻规则

要写清：

- 入驻条件。
- 需要实名审核。
- 需要提交作品和真人资料。
- 服务过程中禁止骚扰、诱导私下交易、泄露用户隐私。
- 平台抽佣比例。
- 结算周期。
- 违规处罚。

#### 退款/取消规则

要写清：

- 用户提前多久取消可退。
- 陪拍者迟到怎么办。
- 用户迟到怎么办。
- 陪拍者爽约怎么办。
- 天气、不可抗力怎么办。
- 服务完成后是否可退款。
- 纠纷如何提交证据。

早期协议不用写得像大公司几十页，但必须覆盖订单、退款、安全、抽佣、违规这些核心问题。

### 12. 招募第一批陪拍者

第一批不要追求数量，追求可控。

建议第一批：

```text
10 - 20 个陪拍者
```

优先招：

- 你认识的人。
- 有真实作品的人。
- 女陪拍者。
- 擅长上海热门路线的人。
- 沟通稳定的人。
- 愿意配合试运营规则的人。

入驻前收集：

- 手机号。
- 身份信息。
- 真人照片。
- 作品样片。
- 服务区域。
- 可服务时间。
- 价格。
- 自我介绍。
- 收款账户。
- 紧急联系人。

早期审核方式可以先人工：

```text
资料提交
-> 视频面试/语音沟通
-> 作品审核
-> 规则确认
-> 小范围开放接单
```

### 13. 设置第一版价格和抽佣

不要一开始价格体系太复杂。

推荐第一版：

```text
1 小时陪拍：199 元
1.5 小时陪拍：299 元
2 小时 Citywalk：399 元
半日陪拍：699 元
```

平台抽佣建议：

```text
15% - 25%
```

第一阶段可以统一：

```text
平台抽佣 20%
```

例子：

```text
用户支付 399 元
平台佣金 79.8 元
陪拍者结算 319.2 元
```

为了财务简单，后台记录时要保留：

- 订单总价。
- 平台佣金。
- 支付通道手续费。
- 陪拍者应结算金额。
- 退款金额。
- 实际结算金额。

### 14. 小范围试运营

第一轮不要公开大规模推广。

建议范围：

```text
10 - 20 个陪拍者
100 - 300 个种子用户
先跑 50 - 100 单
```

试运营目标：

- 用户是否愿意下单。
- 哪些图片最转化。
- 哪些价格能成交。
- 陪拍者是否守时。
- 线下是否安全。
- 退款率高不高。
- 私下交易是否严重。
- 抽佣比例能不能接受。
- 用户复购可能性如何。

第一轮不要追求 GMV，追求发现问题。

每天记录：

- 新用户数。
- 浏览量。
- 咨询数。
- 下单数。
- 支付成功数。
- 完成订单数。
- 退款数。
- 投诉数。
- 陪拍者爽约数。
- 用户爽约数。
- 私下交易风险事件。

## 第二阶段：上线后尽快补

这些不是第一天必须全部做完，但不能长期不做。

### 1. ICP 备案

如果你有官网、H5、后台、接口域名，并部署在中国大陆服务器，需要做 ICP 备案。

准备：

- 公司营业执照。
- 法人信息。
- 域名。
- 云服务器。
- 网站负责人信息。

用途：

- 网站合法访问。
- 小程序业务域名配置。
- App 接口域名使用。
- 应用市场审核。

### 2. 小程序备案 / App 备案

如果先做微信小程序，按微信平台要求完成备案。

如果后续做原生 App，需要做 App 备案，并在 App 内展示备案号。

App 备案通常会涉及：

- 公司主体。
- App 名称。
- 域名。
- 服务内容。
- 负责人信息。
- 平台提交审核。

### 3. 支付分账

当订单量开始稳定后，应尽快接入微信支付或支付宝的分账能力。

目标：

```text
用户支付
-> 平台收取佣金
-> 陪拍者部分按规则结算
-> 资金流更清晰
```

好处：

- 降低资金池风险。
- 财务更清楚。
- 陪拍者更信任。
- 后续规模化更稳。

### 4. 陪拍者实名认证和人脸识别

试运营可以先人工审核，但正式规模化要强化认证。

建议补齐：

- 手机号实名。
- 身份证核验。
- 人脸识别。
- 紧急联系人。
- 黑名单机制。
- 违规记录。

### 5. 完整平台协议

小范围试运营用最简协议，正式运营要补完整版本。

需要补：

- 用户协议。
- 隐私政策。
- 陪拍者服务协议。
- 平台交易规则。
- 内容发布规范。
- 退款和售后规则。
- 举报投诉规则。
- 安全约拍规则。
- 未成年人保护规则。

### 6. 商标注册

尽早注册：

- App 名称。
- Logo。
- 核心品牌名。

优先类别可咨询商标代理，通常要关注互联网服务、软件、广告、摄影、社交/本地生活相关类别。

### 7. EDI / ICP 经营许可证咨询

你的平台如果发展为第三方服务者入驻、用户在线下单、平台抽佣的交易撮合平台，后续可能需要咨询：

- EDI 许可证。
- ICP 经营许可证。

这一步不一定第一天办，但应该在试运营后尽快找专业机构或律师确认。

### 8. App 正式上架

当小程序验证成功后，再做原生 App。

App 上架前准备：

- 公司开发者账号。
- 软著，部分安卓市场可能需要。
- 隐私政策 URL。
- ICP/App 备案。
- 支付 SDK。
- 用户注销功能。
- 权限说明。
- 内容审核机制。
- 投诉举报入口。

### 9. 园区政策和补贴

注册后开始关注杨浦区和园区政策。

重点问：

- 创业培训补贴。
- 创业担保贷款。
- 创业场地支持。
- 科技型中小企业。
- 创新型中小企业。
- 孵化器活动。
- 融资路演。
- 政策申报辅导。

不要为了补贴改变业务本质。补贴是加分项，先把订单跑通。

## 最快 30 天执行表

### 第 1-3 天

- 定公司名称。
- 定法人和股东。
- 定股权比例。
- 找 5-10 个杨浦园区/联合办公询价。
- 确认哪个地址可注册、可开户、可税务核查。

### 第 4-7 天

- 签工位或注册地址合同。
- 准备注册材料。
- 在上海企业登记在线提交公司设立。
- 同步找代理记账。

### 第 8-14 天

- 拿营业执照。
- 拿章。
- 开银行基本户。
- 做税务登记。
- 注册微信小程序账号。
- 注册微信支付商户号。

### 第 15-21 天

- 小程序 MVP 完成基础页面。
- 写最简用户协议、隐私政策、陪拍者规则、退款规则。
- 招募第一批陪拍者。
- 配置价格和抽佣比例。

### 第 22-30 天

- 小范围内测。
- 跑第一批测试订单。
- 记录订单、退款、投诉、私下交易风险。
- 修正价格、规则、页面和流程。
- 准备 ICP/小程序备案资料。

## 第一版不要做的事

为了最快启动，第一版先不要做：

- 全国扩张。
- 多城市运营。
- 大规模广告投放。
- 完整社交社区。
- 复杂 IM 聊天系统。
- 复杂会员体系。
- 自建钱包。
- 自建支付清算。
- 大办公室。
- 大量全职员工。
- 太复杂的补贴申报。

第一版只验证：

```text
用户愿不愿意为陪拍下单
陪拍者愿不愿意接受平台抽佣
平台能不能安全完成订单
这个模式能不能复购
```

## 和外部机构沟通的话术

### 对园区/联合办公

```text
我们准备注册一家网络科技公司，做本地生活预约平台，早期需要 1-2 个工位和可工商注册的地址。想确认地址是否能注册、银行开户、税务核查，以及是否有创业政策辅导。
```

### 对银行

```text
我们是一家本地生活互联网平台，用户可以在线预约城市拍照、探店、Citywalk 等服务。公司主要收入是平台技术服务费和订单服务费，资金通过微信支付等合规支付通道收取。
```

### 对支付平台

```text
我们做本地生活服务预约撮合，用户在线购买摄影陪拍服务，平台收取服务费，服务者完成服务后结算。
```

### 对陪拍者

```text
平台负责获客、展示、订单、支付、规则和售后。你负责按约定完成陪拍服务。平台按订单收取固定比例服务费，剩余金额按结算周期结算给你。
```

### 对用户

```text
平台帮你发现适合的陪拍者，完成预约、支付和售后保障。建议所有交易留在平台内，方便退款、投诉和安全保障。
```

## 最重要的底线

- 不用个人卡收平台订单款。
- 不做自建钱包。
- 不做无牌支付。
- 不长期沉淀服务者资金。
- 不放任私下交易。
- 不允许绕平台交换联系方式交易。
- 不允许陪拍者未经审核接单。
- 不允许没有退款规则就公开收款。
- 不允许没有隐私政策就上线收集用户信息。

## 当前最推荐动作

你现在先做这 5 件事：

1. 确定 3-5 个公司名称。
2. 找杨浦 5 家可注册工位/园区报价。
3. 选一个能注册、能开户、能税务核查的地址。
4. 注册 `上海XX网络科技有限公司`。
5. 同步启动微信小程序 MVP 和微信支付商户号。

等小程序跑通 50-100 单，再开始补完整资质、App 上架和更复杂的政策申报。
